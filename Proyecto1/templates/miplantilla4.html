Como se usa las condicionales.

{# Esta es un condicional if y else#} {% if nombre_persona == 'Luis' %} {# nombre_persona = tomas #}

El curso lo imparte: {{ nombre_persona }}

{% else %}

No hay profesor para este curso

{% endif %} {% comment %} Esto es un comentario de varias lineas {% endcomment %}

================

Sin filtros:

    {% if lista %} {% for tema in lista %}
  • {{ tema }}
    • {% endfor %} {% else %}

    No hay lista

    {% endif %}

================

Con filtro: "|first" Se muestra la primera letra de cada elemento

    {% if lista %} {% for tema in lista %}
  • {{ tema|first }}
    • {# Muestra solo la primera letra de cada palabra #} {% endfor %} {% else %}

    No hay lista

    {% endif %}

================

Con filtro encadenado: "|first|lower" Se muestra la primera letra de cada elemento en minuscula

    {% if lista %} {% for tema in lista %}
  • {{ tema|first|lower }}
    • {# Filtro Encadenado #} {% endfor %} {% else %}

    No hay lista

    {% endif %}
