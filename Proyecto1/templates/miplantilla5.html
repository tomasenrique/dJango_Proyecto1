Trabajando con nuestra quinta plantilla - CARGADORES DE PLANTILLAS

Aqui se vera como cargar una o varias templates de forma mas optima para no usar 'close()' o el 'open()' que son metodos que estan consumiento muchos recursos. Para ello hay que ir al archivo 'setting.py' y buscar la lista 'TEMPLATES' y dentro de el se vera otra lista interna llamada 'DIRS' que aqui se contendra todos los directorios que contengan las paginas web(plantillas - templates) Usando esta forma se ahorra varias lineas de codigo y se optimiza todo ya que aqui ya no se usaria un contexto

Se usa el m√©todo: loader.get_template('miplantilla5.html')

TEMPLATES = [
      {
         'BACKEND': 'django.template.backends.django.DjangoTemplates',
         'DIRS': ["D:/Lenguajes/Django/proyecto1/proyecto1/templates/"], <== Se agrega la carpeta de los templates a usar
         'APP_DIRS': True,
         'OPTIONS': {
              'context_processors': [
                    'django.template.context_processors.debug',
                    'django.template.context_processors.request',
                    'django.contrib.auth.context_processors.auth',
                    'django.contrib.messages.context_processors.messages',
              ],
         },
      },
]

Con filtro: "|first" Se muestra la primera letra de cada elemento

    {% if lista1 %} {% for tema in lista1 %}

================

Con filtro encadenado: "|first|lower" Se muestra la primera letra de cada elemento en minuscula
